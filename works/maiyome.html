What is MaiYome?

MaiYome is "JUST A LOCAL BASED BOORU GALLERY DESKTOP APPLICATION".


Mai is the japanization of the word "My" in english and "Yome" means wife in Japanese, and then combining it become "MaiYome", or something like MaiWaifu / My Waifu

So, basically it's a DAM or Digital Asset Management-like software to manage all of your waifu images collection which focuses on Tag Management just like a booru site but it's locally saved. The UI is very simple, a filter tag, and tag editor to remove or add to your image, also an exporter directly to the image file ( which will change the MD5 ) or just export to .json file.

How to Use?
Import Your Folder
Easy import to the database, just click a button and select the image or the folder, or you can just drag and drop it
Tagging System
Just select an image or batch of image, then search for the tag and apply it, or you can use the preset and apply multiple tags at once, and not just using tags for searching an image, there are other way like md5, file name, and folder to find your favourite image, and you also can import/export your tagged image to or from Json or the image itself
Fingerprinting & Comparator
If you have the same image in multiple folder you can use use fingerprinting, to compare both files, is there any differences or not
