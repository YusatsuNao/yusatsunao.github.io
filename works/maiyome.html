What is MaiYome?

MaiYome is "JUST A LOCAL BASED BOORU GALLERY DESKTOP APPLICATION".


Mai is the japanization of the word "My" in english and "Yome" means wife in Japanese, and then combining it become "MaiYome", or something like MaiWaifu / My Waifu

So, basically it's a DAM or Digital Asset Management-like software to manage all of your waifu images collection which focuses on Tag Management just like a booru site but it's locally saved. The UI is very simple, a filter tag, and tag editor to remove or add to your image, also an exporter directly to the image file ( which will change the MD5 ) or just export to .json file.

How to Use ?
Importing Your Image/Folder

1. Just click the IMPORT button
2. Select the Image or the Folder you want to import
3. And there you go, the images are ready to be tagged

Tagging / Preset Tagging

1. Select your Image/Multiple Image from the list
2. in the Apply TAG, search and input the tag you want to add and then press Enter
or you can use preset by typing Preset:{your preset name} in the Apply TAG Box
3. And then all Images you select are tagged automatically
4. If you want to search your favourite image, you can search it using the TAG, MD5, FileName, and Folder Name
5. and you also can Import/Export your tagged image to or from Json or the image itself

Fingerprinting & Comparator

If you have the same image but in different folder, you can use Fingerprinting to compare both images, is there any differences or not
